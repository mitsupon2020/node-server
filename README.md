## これはなに？
Node.js環境で動作する簡易サーバです。


## 想定利用
- 簡易的なmockサーバ
- 本格的な機能を実装前のテンプレート

## 実装機能

- 予定
　- 静的ファイル返却
　- templateエンジンを利用し、htmlを返却
  - ルーティング
  - 認証付きコンテンツ配信
  - dynamodb

## 動かし方

```
git clone 
npm install
npm run serve
```
